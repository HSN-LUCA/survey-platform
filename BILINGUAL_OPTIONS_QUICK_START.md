# Bilingual Options - Quick Start

## What's New?

You can now create survey options in both English and Arabic with preset templates!

## Quick Example

### Before (Old Way)
```
Options: Option 1, Option 2, Option 3
```
- Only one language
- Manual typing
- No consistency

### After (New Way)
1. Select "Satisfaction" preset
2. Get 5 options in both languages automatically:
   - Very Satisfied / راضٍ جداً
   - Satisfied / راضٍ
   - Somewhat Satisfied / راضٍ إلى حد ما
   - Dissatisfied / غير راضٍ
   - Very Dissatisfied / غير راضٍ جداً

## How to Use (3 Steps)

### Step 1: Create Survey
Admin Dashboard → Create Survey

### Step 2: Add Question
- Click "Add Question"
- Fill in question text (English & Arabic)
- Select "Multiple Choice"

### Step 3: Load Options
- Scroll to "Use Preset Options"
- Select a category:
  - Agreement
  - Yes/No
  - Satisfaction
  - Likelihood
  - Quality
  - Frequency
  - Importance
  - Recommendation
  - Service Quality
- Options load automatically!

## Available Presets

| Category | Options | Use Case |
|----------|---------|----------|
| Agreement | 5 options | "Do you agree?" |
| Yes/No | 3 options | "Would you?" |
| Satisfaction | 5 options | "How satisfied?" |
| Likelihood | 4 options | "How likely?" |
| Quality | 4 options | "Quality rating?" |
| Frequency | 5 options | "How often?" |
| Importance | 4 options | "How important?" |
| Recommendation | 2 options | "Recommend?" |
| Service Quality | 4 options | "Service rating?" |

## Add Custom Options

If presets don't fit:

1. Load a preset (or start empty)
2. Scroll to "Or Add Custom Option"
3. Enter English text
4. Enter Arabic text
5. Click "+ Add Option"
6. Repeat for more options

## Remove Options

- Click "Remove" on any option to delete it
- Click "Clear All Options" to start over

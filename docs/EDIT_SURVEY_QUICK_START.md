# Edit Survey - Quick Start Guide

## What's New

You can now edit existing surveys! Modify survey details, questions, and options after creation.

## How to Edit a Survey

### Method 1: From Survey List
1. Go to **Admin Dashboard** ‚Üí **Surveys**
2. Find the survey you want to edit
3. Click the **Edit** button (green button)
4. Make your changes
5. Click **Update Survey**

### Method 2: From Survey Detail
1. Go to **Admin Dashboard** ‚Üí **Surveys**
2. Click **View Survey** on any survey
3. Click **Edit Survey** button in the header
4. Make your changes
5. Click **Update Survey**

## What You Can Edit

### Always Editable
- Survey title (English & Arabic)
- Survey description (English & Arabic)
- Customer type (Pilgrims/Staff)

### Editable Only If No Responses
- Questions (add, edit, remove)
- Question types
- Question options
- Question categories
- Required field status

### If Survey Has Responses
- ‚ö†Ô∏è Questions cannot be edited
- ‚úÖ Title and description can still be edited
- ‚úÖ You'll see a warning message

## Editing Questions

### Add a New Question
1. Click **+ Add Question** button
2. Enter question text in English and Arabic
3. Select question type
4. Add options (for multiple choice)
5. Click **Update Survey**

### Edit Existing Question
1. Modify question text
2. Change question type (if no responses)
3. Update options
4. Click **Update Survey**

### Remove a Question
1. Click **Remove** button on the question
2. Click **Update Survey**

## Editing Options

### Add Option
1. Select preset options from dropdown, OR
2. Enter custom option in English and Arabic
3. Click **+ Add Option**
4. Click **Update Survey**

### Remove Option
1. Click **Remove** button on the option
2. Click **Update Survey**

### Use Preset Options
1. Click **Use Preset Options** dropdown
2. Select a category (Agreement, Yes/No, Satisfaction, etc.)
3. Options are automatically loaded
4. Click **Update Survey**

## Important Notes

‚ö†Ô∏è **Survey with Responses**
- If your survey has responses, you can only edit the title and description
- Questions cannot be changed to maintain data integrity
- You'll see a yellow warning message

‚úÖ **Best Practices**
- Always provide content in both English and Arabic
- Multiple choice questions need at least one option
- Use categories to group related questions
- Test your survey before sharing

## Troubleshooting

### "Survey not found"
- The survey ID is invalid or the survey was deleted
- Go back and try again

### "Unauthorized"
- Your session expired
- Log in again

### "Failed to update survey"
- Check that all required fields are filled
- Ensure questions have content in both languages
- Multiple choice questions must have options

## Tips

üí° **Preset Options**
- Use preset options to save time
- Available categories: Agreement, Yes/No, Satisfaction, Likelihood, Quality, Frequency, Importance, Recommendation, Service Quality

üí° **Categories**

üí° **Bilingual Support**
